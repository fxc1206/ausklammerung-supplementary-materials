## 章节速览

### B.1 · 破框分布统计

- 识别流程：基于 UD_German-HDT v2.13（153,035 句），依据拓扑场域模型自动提取后场成分
- 人工核验：分层随机抽样 200 例（正/负各 100），识别精确率 **95%**（95% CI [88.7%, 98.4%]）
- 分布结果：总体破框率 **12.21%**；非短语形式占 88.84%，从句最多（59.60%）

### B.2 · 特征筛选过程

- 起点：18 项候选指标分属拓扑 / 加工 / 信息 / 形式 / 角色 五维
- 共线性筛除：Pearson |r| ≥ 0.70 阈值，识别 3 组共线性
- 互信息筛选：各维度 MI 排名前两位 → 保留 9 项核心特征
- 显著性确认：Mann-Whitney U 检验 + rank-biserial 效应量

### B.4 · Causal SHAP 路径归因

- 算法：基于 mGPT 的 surprisal 估计 + PC 算法构建 DAG
- 路径分解：直接贡献 vs 经其他特征的间接（传导）贡献
- 分类阈值：传导占比 ≥ 70% → 传导主导；直接占比 ≥ 70% → 直接主导
